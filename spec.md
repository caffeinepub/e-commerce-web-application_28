# Specification

## Summary
**Goal:** Pre-populate the Shop ICP application with at least 10 diverse sample products so users can immediately browse and place orders.

**Planned changes:**
- Add at least 10 sample products to the backend initialization, spanning at least 3 categories (e.g., Electronics, Clothing, Home & Kitchen), each with a name, description, category, price, stock quantity, and a publicly accessible image URL.
- Update the `SeedProducts` component in the frontend to include the same expanded set of sample products, so admins can re-seed the catalog when needed.

**User-visible outcome:** Users can immediately browse a populated product catalog with 10+ products across multiple categories and add them to the cart and place orders without any manual data entry. Admins can also re-seed the catalog via the Admin Dashboard.
